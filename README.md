## 2. 实现点击事件
### 相关变量
+ 正方形canvas 
+ 有一个空白边（padding）
+ 棋盘格子的长度（interval）*3=棋盘边长  
+ 棋子半径（r）  
+ 棋子圆心坐标 ```(padding+interval*n,padding)```或```(padding+interval*n,padding+interval*3)```  
### 移动棋子
1. 获取鼠标坐标（x,y）
2. 判定在哪一个子的范围内，以棋子r为边长的正方形范围
3. 精确判断，是否在圆内，如果是，返回坐标
4. 使用正方形判定范围，判断移动到新的点上
5. 更新数组，重绘
> 需要注意：
> + 新位置上是否有棋子
> + 是否只移动了一格
